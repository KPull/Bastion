Get Bastion using Maven

Also read our GitHub readme for more examples.

<dependency>
    <groupId>rocks.bastion</groupId>
    <artifactId>bastion</artifactId>
    <version>[VERSION]</version>
    <scope>test</scope>
</dependency>

JSON Response Assertions

Bastion quickly lets you type in the JSON response you're expecting. The order of properties does not matter (as in the JSON spec) and Bastion will even print the error in JSON Patch format to tell you how to transform the actual response to the expected one.

[ex:simple-json-assertion]

Bind Data

Use the bind() method to automatically parse HTTP entity data into a Java object for use later on in the test.
